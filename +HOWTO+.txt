To UPDATE:
> yt-dlp -U


Simple Video Download (will automatically download as mp4 and use 'node' for the js runtime, per the configuration file):
> yt-dlp [URL]

** NOTE:  Firefox cookies are now used automatically (set as a default in yt-dlp.conf as "--cookies-from-browser firefox") because without them YouTube returns 403 errors.
** If Firefox is open and yt-dlp complains about a locked cookie DB, close Firefox and re-run.
** If the default profile isn't picked up, override with an explicit path:  yt-dlp --cookies-from-browser firefox:"Path/to/your/profile" URL


To download playlist:
> yt-dlp.exe --download-archive downloaded.txt --output "%(playlist_title)s/%(title)s.%(ext)s" "https://www.youtube.com/playlist?list=PLAYLIST_ID"



---------
NOTE:  If you hit rate limiting issues, then remove or comment out this line from the config file:
>> -N 4
(This line just tells the application to use 4 concurrent downloads, which is useful for YouTube, but MAY cause rate limiting issues on other sites.)
